# 📱 Ótica Sistema — Como publicar no GitHub Pages

## Passo 1 — Criar conta no GitHub
1. Acesse https://github.com e clique em **Sign up**
2. Crie uma conta gratuita (pode usar qualquer e-mail)

## Passo 2 — Criar repositório
1. Depois de entrar, clique no **+** no canto superior direito
2. Clique em **New repository**
3. Nome do repositório: `otica-sistema`
4. Marque como **Public**
5. Clique em **Create repository**

## Passo 3 — Subir os arquivos
1. Na página do repositório criado, clique em **uploading an existing file**
2. Arraste TODOS os arquivos desta pasta:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Clique em **Commit changes**

## Passo 4 — Ativar GitHub Pages
1. Vá em **Settings** (no menu do repositório)
2. No menu lateral, clique em **Pages**
3. Em "Branch", selecione **main** e clique **Save**
4. Aguarde 1-2 minutos

## Passo 5 — Acessar o sistema
Seu link será:
```
https://SEU_USUARIO.github.io/otica-sistema/
```

## 📲 Instalar como app no celular

### Android (Chrome):
1. Abra o link no Chrome
2. Toque nos **3 pontinhos** no canto superior
3. Toque em **"Adicionar à tela inicial"**
4. Confirme — pronto! Ícone aparece como app!

### iPhone (Safari):
1. Abra o link no Safari
2. Toque no botão de **Compartilhar** (quadrado com seta)
3. Toque em **"Adicionar à Tela de Início"**
4. Confirme — pronto!
